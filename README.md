# 🔭 Astra — KP Astrology App

A personal astrology app built on the **Krishnamurti Paddhati (KP)** system, using Swiss Ephemeris for precise planetary calculations.

## Features

- 🪐 **Planetary positions** — sign, nakshatra, pada, star lord, sub lord, sub-sub lord
- 🏠 **House cusps** (Placidus/KP) with sublords
- 🎯 **Significators** — 3 levels of KP house significators
- ⏳ **Vimsottari Dasha** — Maha, Antardasha & Pratyantar with exact dates
- 📐 **Divisional charts** — D1 through D60
- 🌍 **Current transits** & ruling planets
- 💾 **Save & load** multiple charts

## Run Locally

```bash
pip3 install -r requirements.txt
python3 -m streamlit run app.py
```

Then open http://localhost:8501 in your browser.

## Ayanamsa Options

- KP (Krishnamurti) — default
- Lahiri
- Raman
